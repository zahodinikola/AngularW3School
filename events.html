The event directives allow us to run AngularJS functions on certain user events

An AngularJS event will not overerite an HTML event, both events will be executed


Mouse events


Mouse event occur when the cursor moves over an element, in this order

  1. ng-mouseenter
  2. ng-mouseover
  3. ng-mouseleave
  4. ng-mousemove

Or when the mousebutton is clicked on an element. In this order:

  1. ng-mousedown
  2. ng-mouseup
  3. ng-click

Mouse events can be added on any HTML element
