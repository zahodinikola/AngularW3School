AngularJS ng-model directive


The ng-model directive binds the value of HTML controls (input, select, textarea) to application data

The ng-model directive

and two way binding

With the ng-model directive you can bind the value of an input field to a variable created in AngularJS

Validate user input

The ng-model directive can provide type validation for application data (number, email, required).

The span element will be displayed only if expression of ng-show attribute returns true

If the property in the ng-model attribute doesn't exist, AngularJS will create it for you

Application status


The ng-model directive can provide status for application data (invalid, dirty, touched, error)

CSS classes

The ng-model directive provides CSS classes for HTML elements, depending on their status
