AngularJS controllers


AngularJS applications are controlled by controllers

The ng-controller directive defines the application controller

A controller is a JavaScript Object created by a standart JavaScript object constructor

The AngularJS application is defined by ng-app="myApp". The application runs inside the <div>. The ng-controller="myCtrl" attribute is an AngularJS directive. It defines a controller. The myCtrl funstion is a JavaScript function. AngularJS will invoke the controller with a $scope object. In AngularJS $scope is the application object (the owner of application variables and functions). The ng-model directives bind the input fields to the controller properties.
