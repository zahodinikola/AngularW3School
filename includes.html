AngularJS Includes

With AngularJS you can include HTML content using ng-include directive:

                <body ng-app="">
                    <ng-include="'file.htm'">
                </body>

Include cross domains

By default the ng-include directive doesn't allow you to include files fron other domains

To include files from another domain you can add a whitelist of legal files and/or domains in the config function of your application

Be sure that the server on the destination allows cross domain access
