AngularJS animations

AngularJS provides animated transitions with help from CSS

What is an animation?

An animation is when the transformation of HTML element gives you an illusion of motion


Hide the div


Applications shouldn't be filled with animations, but some animations can make the application easier to understand

What do i need?

To make your application ready for animations you must include AngularJS Animate library

			<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.4/angular-animate.min.js"></script>

Then you must refer to the ngAnimate module in your application

			<body ng-app="ngAnimate">

Or, if you application has a name, add ngAnimate as a dependency in your application module
