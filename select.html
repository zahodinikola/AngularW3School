AngularJS Select boxes

AngularJS lets you create dropdown lists based on arrays or an object

Creating a select box using ng-options

If you want to create a dropdown list based on an array or an object in AngularJS you should use the ng-options directive

ng-options vs ng-repeat

You can also use the ng-repeat directive to make the same dropdown list

The ng-repeat directive repeates the block of HTML code for each item in an array

It can be used to create options in a dropdown list, but the ng-options directive was made specially for filling a dropdown list with options and has at least one importante advantage:

dropdowns made with ng-options allows the selected value to be an object, while dropdowns made from ng-repeat has to be a string


What do i use?


Lets assume we have an array of objects

                $scope.cars = [
                    {model : "Ford Mustang", color : "red"},
                    {model : "Fiat 500", color : "white"},
                    {model : "Volvo XC90", color : "black"}
                ];

The ng-repeat directive has its limitations, the selected value must be a string

When using the ng-options directive, the selected value can be an object:

When the selected value can be an object, it can hold more information and your appllication can be more flexible


The Data Source as an Object

In the previous examples the data source was an array, but it can also be an object

                $scope.cars = {
                    car01 : "Ford",
                    car02 : "Fiat",
                    car03 : "Volvo"
                };

The expression in ng-options attribute is a bit different for objects

The selected value will always be the value in a key-value pair

The value in a key-value pair can also be an object

The options in the dropdown list can also be the value or a property of the value object
