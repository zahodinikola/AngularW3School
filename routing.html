AngularJS routing


The ngRoute module helps your application to become a Single Page Application


What is routing in AngularJS ?


If you want to navigate to different pages in your application, but you also want your application to be a SPA (Single Page Application), with no page reloading, you can use ngRoute module

The ngRoute module routes your application from page to page without reloading entire application

Example: navigate to red.htm, green.htm and blue.htm
